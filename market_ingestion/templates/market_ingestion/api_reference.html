Query Parameters

Supported Filters

Use query parameters to narrow the list endpoints.

  • source — provider name, such as yfinance
  • symbol — instrument symbol, such as BTC-USD
  • timeframe — interval value, such as 1h
  • status — available only on ingestion runs, such as success
  • page — page number for paginated results
  • page_size — page size, capped in the backend
Runs: status=success Runs: source=yfinance Snapshots: symbol=BTC-USD Bars: yfinance + BTC-USD + 1h Bars: page=1&page_size=25
