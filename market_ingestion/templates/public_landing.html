{# ── CAPABILITIES ─────────────────────────────────────────────────────────── #}
What It Demonstrates

Core Capabilities

Built to show the full span from data engineering fundamentals to analytics delivery and operational observability.

01
⚡
Multi-Source Ingestion
Connects to yfinance, ccxt, and TwelveData through reusable client modules and a structured ingestion flow.
02
🗄️
Normalized Persistence
Provider responses are transformed into structured market bars, ETL run records, snapshots, and journal-linked analytics context.
03
🔁
Repeatable ETL Workflows
Three management commands handle ingest, journal import, and metric computation through inspectable terminal-driven workflows.
04
📊
Metrics Computation
Computes reusable snapshot outputs — returns, volatility, moving averages, and crossover signals — for reporting and review.
05
🔍
Operational Monitoring UI
SaaS-style internal views for reviewing ETL run history, stored bars, snapshots, and platform state with filter support.
06
🔌
Read-Only API Layer
JSON endpoints under /api/ops/ expose runs, snapshots, bars, and journal entries with query parameter filtering.

Schema Design

Data Models

Normalized relational models designed for downstream analytics querying, reporting, and operational inspection.

IngestionRun
  • source CharField
  • symbol CharField
  • timeframe CharField
  • status CharField
  • rows_inserted / rows_updated / rows_skipped IntegerField
  • metadata JSONField
  • started_at / finished_at DateTimeField
MarketBar
  • ingestion_run ForeignKey
  • source / symbol / timeframe CharField
  • open / high / low / close DecimalField
  • volume DecimalField
  • timestamp DateTimeField
  • raw_payload JSONField
MetricSnapshot
  • symbol / timeframe CharField
  • snapshot_time DateTimeField
  • close_price DecimalField
  • return_1d / return_7d DecimalField
  • volatility_14d DecimalField
  • sma_fast / sma_slow DecimalField
  • crossover_signal CharField
TradeJournalEntry
  • symbol / side CharField
  • opened_at / closed_at DateTimeField
  • open_price / close_price DecimalField
  • size / pnl DecimalField
  • source_file / row_number CharField / IntegerField
  • notes TextField
  • raw_row JSONField
